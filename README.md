## Local Development :-
- `git clone https://github.com/DogeGram/Frontend.git`
- `cd Frontend`
- Make a .env file and type `REACT_APP_BACKEND_URL=http://localhost:5000`
- `npm install`
- `npm start_nossl`
Your frontend server should start at port `3000`
